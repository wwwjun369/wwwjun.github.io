听说你也睡不着

发表于 2017-11-23 更新于 2019-08-15 分类于 生活 ， 推荐 阅读次数：
本文字数： 1.9k 阅读时长 ≈ 2 分钟

习惯了晚睡，晚起，有时候甚至会失眠到几点。有时候很累很困，但是一回到家，打开电脑却一点也不困了，有时候躺床上听了好久的歌曲也没睡着。

慢慢地开始探索了好多电台节目等；刚折腾好博客也不知道写些什么，就来推荐下我收藏的一些电台、主播吧。

  • 睡不着电台
    网易云音乐搜索或者点击下面的链接，白无常白总的声音真的很好听，很治愈，有点类似台湾女生那样子。
    http://music.163.com/#/user/home?id=246202585
  • 晓苏电台
    网易云音乐搜索或者点击下面的链接，晓苏的声音也是很好听，讲得大部分也都是情感故事。Hey 你好吗？我是晓苏……
    http://music.163.com/#/radio/?id=203&userid=48170583
  • 蕊希电台
    网易云音乐搜索或者点击下面的链接，同样的就不啰嗦了。
    http://music.163.com/#/radio/?id=618058&userid=48170583

2. 心理FM

接下来推进一下很久以前我就接触的心理FM
应该是上大学那会，或者高三？不太记得具体日期了，但是有句话却很印象深刻，那就是：世界和我爱着你，那时候就喜欢听阡语陌路主播的，现在好像她很少在心理FM更新了，微信公众号：阡陌对你说晚安(Lisa_voice)
还有两个桑田和潇湘也不错，很久没有听其他人的了或许你们可以自己去发现美的声音。

  • 下雨声

4. ASMR

最后这个厉害了，传说中的颅内高潮（捂脸ing）……其实就是掏耳朵而已哈哈哈

ASMR (Autonomous Sensory Meridian Response)，中文译名“自发性知觉经络反应”，是一个用于描述感知现象的新词，其特征是：对视觉、听觉、触觉、嗅觉或者感知上的刺激而使人在颅内、头皮、背部或身体其他范围内产生一种独特的、令人愉悦的刺激感。(摘自百度百科）

简单点讲就是用各种东西弄出来声音，这些声音让你听了很舒服，放松之类的，有些声音很像别的声音然后你主观意识下就很放松等等，种类也很多。可以自行百度下。

现在好像也挺火的，好像很多直播平台现在也有很多这样的直播，很好地结合了类似哔哩哔哩这种二次元御宅文化等。
