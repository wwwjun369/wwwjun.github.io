Get It Right

发表于 2018-03-18 更新于 2019-08-15 分类于 生活 阅读次数：
本文字数： 378 阅读时长 ≈ 1 分钟

按照新历算的话，2018年不知不觉已经过了三个多月了，虽然我对我来说一般过了春节才算是新的一年的开始，然而春节也过去了好久了。前段时间一直想写个总结和新的一年的机会来着，但是颓废了一段时间，懒。

过去的一年发生了很多事情，无论工作上还是生活上，不能说收获很大吧，但是还是成长了。只是希望自己今年好好学习积累，过得更好吧。

现在不是很想去写一些东西，还没想好怎么写，还是先写下我今年要做的事情吧：

1.报驾校，学车
2.坚持健身，运动
3.提高专业技能
4.规律作息，早睡早起，不失眠，注意养生
5.少说一些欠打的话，可以幽默但尽量不要建立在别人的痛苦之上哈哈
6.练习普通话发音，把容易读错或者读不准的词记下来
7.学习理财，基金股票等
8.多吃水果,多喝水，喝茶
9.有些病得治，至少弄清楚如何去改善
10.坚持学习摄影
11.再长点肉

Get it right~
